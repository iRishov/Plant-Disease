TensorFlow

TensorFlow is an open-source framework by Google for building and deploying ML models at scale, supporting deep learning and neural networks.

TensorFlow Lite

TensorFlow Lite enables model deployment on mobile and edge devices with optimized performance and low memory footprint.

FastAPI

FastAPI is a modern, high-performance Python web framework for building APIs with automatic OpenAPI and JSON schema generation.

Plant Disease Dataset

Plant Disease Dataset provides labeled images of plant leaves to train ML models in detecting diseases.

Custom CNN

A custom Convolutional Neural Network (CNN) designed for fast and efficient plant disease detection. Ideal for real-time or resource-limited scenarios.

EfficientNetV2S

EfficientNetV2S is a fast and accurate convolutional neural network optimized for both training and inference speed, offering an excellent balance of accuracy and efficiency.

InceptionResNetV2

InceptionResNetV2 combines the power of Inception modules with residual connections for state-of-the-art accuracy, best suited for research or critical diagnoses where accuracy is paramount.

ResNet50V2

ResNet50V2 introduces improved residual connections to increase model accuracy in image classification tasks.

Machine Learning

Machine learning enables AgroAI to learn from plant data, identify disease symptoms, and make accurate predictions using model inference.
